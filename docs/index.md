# CamTrap Verify

![WildINTEL](img/wildIntel_logo.webp){ style="display: block; margin: 0 auto;" }

**CamTrap Verify** is an open-source web application that helps ecologists and wildlife researchers review and validate species detections produced by AI classifiers or citizen-science platforms.

It reads any [CamtrapDP v1.0](https://camtrap-dp.tdwg.org/) directory and guides the expert through a structured, iterative review that minimises the number of images that must be inspected.
